Die Klassen stellen eine (hoffentlich verständliche) Aufrufhülle zur Verfuegung
fuer die wichtigen zum Zeichnen benoetigten Klassen, die das Interface Shape
implementieren.

Die Parabel_2 ist beispielhaft eingefuegt, ist aber fuer schulische Zwecke
wohl eher nicht geeignet. Verwendet wird eine "Bézier-Kurve".
Insbesondere fuer die nicht realisierte Parabel 3. Grades sollte man vorher
bei z.B. Wikipedia nachlesen, wie sie zu verwenden sind.
